== Description ==

This WordPress plugin will give the best way to keep up with your content feed by placing a feedblitz email subscription on your wordpress site.
Feedblitz Email Subscription gives the best way to keep up with your content feed by placing a feedblitz email subscription on your wordpress site.
Easy to customize, just put your feed id and publisher id and your ready to lunch.
Shortcode usage:<strong>[feedblitz id="your_feed_id" publisher_id="your_publisher_id"]</strong>.

== Installation ==

<p>Here's how to use Feedblitz Email Subscription:</p>
<ol>
<li><a href = "http://wordpress.org/plugins/feedblitz-email-subscription/" >Download </a>the plugin.</li>
<li>Upload the plugin zipped file to your desired wordpress site.</li>
<li>Activate the plugin.
<li>To use the widget, go to widget area look for - 'Feedblitz Email Subscription'.</li>
<li>To use the shortcode, go to the page where you want to add the shortcode , insert the code - <strong>[feedblitz id="your_feed_id" publisher_id="your_publisher_id"]</strong>.
<br/><br/>
***NOTE*** : <br/>
	These two values the 'id' and 'publisher_id' are very neccessary. <br/>
	The 'id' refers to your 'your_feed_id', and the other id refers to your "your_publisher_id".<br/><br/>

YOU CAN GET THESE TWO ID's BY FOLLOWING THESE STEPS:

<ol>
  <li>LOGIN to your Feedblitz account. Refer to this link => https://www.feedblitz.com</li>
  <li>Select and click the site where you want to put the feed.</li>
  <li>Look for the 'FORMCODE' button in the displayed thumbnails and CLICK IT.</li>
  <li>After that an html form will be showed,,,look for these two input field:
     <ul>
	<li>&lt;input name=&quot;FEEDID&quot; type=&quot;hidden&quot; value=&quot;11111&quot;&gt;</li>
	<li>&lt;input name=&quot;PUBLISHER&quot; type=&quot;hidden&quot; value=&quot;2222222&quot;&gt;</li>
    </ul>
  </li>
</ol>	
</li>
<li>Copy and paste each value respectively to your shortcode.</li>
<li>Finished.</li>
</ol>
